# Basic operations and hotkeys of `screen`

1. screen -S name

- create a screen named `name`

2. screen -ls

- list all screens

3. screen -r name

- resume a screen named `name`

4. Ctrl + a + d

- left a screen

5. Ctrl + d

- kill the current screen

6. Ctrl -D -r name

- resume an attached screen

7. Ctrl + a + esc

- enter the copy mode

8. screen -wipe

- delete unused sceen
